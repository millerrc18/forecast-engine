Getting Started

To begin forecasting, follow these steps:

  1. Import actuals data — Upload a CSV/Excel export from Unanet with hours by program, period, cost pool, and person
  2. Create a demand signal — Tell functional managers what you expect next quarter (units, milestones, scope changes)
  3. Functional managers respond with staffing allocations — FTEs, allocation %, and rates per cost pool
  4. Generate forecasts — The system calculates Tier 1 (headcount-based) estimates and, once the ML model is trained, Tier 2 (Gradient Boosting) predictions with SHAP explainability
